Em desenvolvimento
Em desenvolvimento
E-commerce Painel admin Mercado Pago Firebase
Stack usada
  • Front-end modular em JavaScript (ES6+)
  • Back-end Node.js + Express
  • Firebase Auth + Firestore + Admin SDK
  • Mercado Pago (Pix, boleto e cartão)
  • Build com esbuild + otimização de imagens
O que foi feito (loja)
  • Catálogo com filtros, busca e organização por categorias
  • Home com coleções (retrô, jogador e destaques)
  • Página de produto com variações e CTA de compra
  • Carrinho lateral + carrinho completo
  • Checkout com Pix, boleto e cartão
  • Rastreio de pedidos com timeline
  • Cupons, recompensas e newsletter
O que foi feito (admin)
  • Dashboard com usuários, produtos, pedidos, leads e receita
  • Gestão de catálogo, pedidos, cupons e recompensas
  • Configuração de site e módulos operacionais
  • Moderação de perguntas e mensagens de contato
  • Leitura de funil e métricas para decisão comercial
Engenharia e segurança
  • Autorização admin por claims/token + validação Firestore
  • Rate limiting por domínio (checkout, admin, analytics, newsletter)
  • Endpoints de pagamento + webhook + status do pedido
  • Notificações por e-mail em eventos de pedido/pagamento
  • Base preparada para escalar e evoluir módulos
Etapa atual

Finalização dos módulos administrativos restantes, refino de performance e ajustes finais para release comercial completo.
